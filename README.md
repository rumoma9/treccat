# treckcat

https://rumoma9.github.io/treccat/

KML / GPX ファイルを読み込み、ルートをマップ上でアニメーション再生するウェブアプリです。

## 機能

- KML / GPX ファイルを最大 4 ファイルまで読み込み、同時再生
- OpenStreetMap ベースのマップ上にルートを表示
- 高度プロファイルチャート（クリックで任意地点にジャンプ）
- 自転車モード / ウォーキングモード（ノイズ除去設定が異なる）
- 再生速度の倍率変更（10x〜1000x）
- 経過時間・走行距離・速度のリアルタイム表示

## 使い方

1. モード切替ボタンで用途を選択する（自転車 / ウォーキング）
2. アップロードボタン（ピンク）で `.kml` または `.gpx` ファイルを選択
3. 倍速（Rate）を選択する
4. 再生ボタン（▶）をクリック

詳細は [操作マニュアル](https://rumoma9.github.io/treccat/manual.html) を参照してください。

## 画面構成

| エリア | 説明 |
|--------|------|
| マップ（上部） | OpenStreetMap ベースの地図。ドラッグでパン、スクロールでズーム |
| チャート（下部） | ルートの高度プロファイル。X 軸: 距離 (km)、Y 軸: 高度 (m) |
| コントロールパネル（右上） | 再生・停止・ファイルアップロードなどの操作ボタン |
| データ表示（左下） | 選択中サイクリストの経過時間・走行距離・速度 |

## 使用ライブラリ

- [OpenLayers](https://openlayers.org/) — 地図表示
- [C3.js](https://c3js.org/) / [D3.js](https://d3js.org/) — チャート描画
- [Fomantic UI](https://fomantic-ui.com/) — UI コンポーネント
